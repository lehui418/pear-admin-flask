## 系统登录

### 1. 访问系统

在浏览器地址栏输入系统地址：`http://服务器地址:5000`

### 2. 输入账号密码

- **默认管理员账号**：`admin`
- **默认密码**：`123456`

### 3. 输入验证码

输入图片中显示的验证码，点击"登录"按钮。

### 4. 记住密码

勾选"记住密码"可在下次访问时自动填充登录信息。

## 工单管理

### 工单列表

#### 查看工单列表

1. 点击左侧菜单"工单管理" → "工单记录"
2. 系统显示所有工单列表
3. 列表包含以下信息：
   - 工单ID
   - 工单标题
   - 优先级
   - 工单状态
   - 负责人
   - 创建时间
   - 更新时间
   - 完成时间

#### 搜索工单

1. 点击页面右上角的搜索图标（🔍）
2. 展开搜索区域
3. **常用筛选**：
   - 工单标题/ID：输入关键词搜索
   - 优先级：选择 P1/P2/P3/P4
   - 序列号：输入设备序列号
   - 负责人：输入负责人姓名
   - 客户/代理商：输入客户名称
   - 工单状态：选择当前状态
4. 点击"搜索"按钮执行搜索
5. 点击"重置"清空筛选条件

#### 高级筛选

1. 点击"高级筛选"展开更多筛选条件
2. **产品信息**：
   - 产品类型
   - 产品二级分类
   - 版本号
   - 是否过保
3. **服务与处理信息**：
   - 响应方式
   - 影响范围
   - 接单时间
   - 完成时间
4. **问题分析与处理过程**：
   - 问题分类
   - 模块标签
   - 处理记录
   - 处置方案

### 创建工单

#### 方式一：快速创建

1. 点击"新增工单"按钮
2. 填写基本信息：
   - **工单标题**（必填）：简要描述问题
   - **问题描述**（必填）：详细描述问题现象
   - **优先级**（必填）：选择 P1-P4
3. 填写产品信息（可选）：
   - 产品类型
   - 序列号
   - 版本号
4. 点击"提交"按钮

#### 方式二：完整创建

1. 点击"新增工单"按钮
2. 填写所有必填项
3. 填写选填项以获得更完整的记录
4. 上传相关附件（如截图、日志等）
5. 点击"提交"按钮

### 查看工单详情

1. 在工单列表中找到目标工单
2. 点击操作列的"查看"按钮（👁️图标）
3. 查看工单详细信息：
   - 基本信息
   - 产品信息
   - 服务与处理信息
   - 问题分析与处理过程
   - 工单进度时间线

### 编辑工单

#### 编辑条件

只有以下人员可以编辑工单：
- 工单创建者
- 当前负责人
- 管理员
- 研发部门成员（针对研发相关工单）

#### 编辑步骤

1. 找到目标工单
2. 点击操作列的"编辑"按钮（✏️图标）
3. 修改需要更新的字段
4. 点击"保存"按钮

#### 状态变更

编辑工单时可以变更状态：
- **创建/提交** → 工单刚创建
- **处理中** → 开始处理
- **已解决** → 问题解决
- **已关闭** → 工单关闭
- **暂时规避** → 临时解决方案

### 删除工单

#### 删除条件

- 工单创建者可以删除自己创建的工单
- 管理员可以删除所有工单
- 负责人可以删除自己负责的工单

#### 删除步骤

1. 找到目标工单
2. 点击操作列的"删除"按钮（🗑️图标）
3. 确认删除对话框
4. 点击"确定"删除

#### 批量删除

1. 勾选需要删除的工单（左侧复选框）
2. 点击工具栏"删除选中"按钮
3. 确认删除
4. 点击"确定"批量删除

### 导出工单

#### 导出所有

1. 点击"导出所有"按钮
2. 系统自动下载 Excel 文件
3. 文件包含所有工单数据

#### 导出筛选结果

1. 先进行筛选搜索
2. 点击"导出所有"按钮
3. 下载的 Excel 只包含筛选后的数据

## 系统管理

### 用户管理

#### 查看用户列表

1. 点击左侧菜单"系统管理" → "用户管理"
2. 查看所有系统用户
3. 显示信息：用户名、真实姓名、部门、角色、状态

#### 新增用户

1. 点击"新增"按钮
2. 填写用户信息：
   - 用户名（必填）
   - 密码（必填）
   - 真实姓名
   - 部门
   - 角色
3. 点击"提交"

#### 编辑用户

1. 找到目标用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 点击"保存"

#### 禁用/启用用户

1. 找到目标用户
2. 点击"禁用"或"启用"按钮
3. 确认操作

### 部门管理

#### 查看部门结构

1. 点击"系统管理" → "部门管理"
2. 查看部门树形结构
3. 支持展开/收起子部门

#### 新增部门

1. 点击"新增"按钮
2. 填写部门名称
3. 选择上级部门
4. 点击"提交"

### 角色权限

#### 查看角色

1. 点击"系统管理" → "角色管理"
2. 查看所有角色列表
3. 显示：角色名称、角色代码、状态

#### 分配权限

1. 找到目标角色
2. 点击"权限"按钮
3. 勾选需要分配的权限
4. 点击"保存"

## 常见问题

### Q1: 忘记密码怎么办？

**A**: 联系系统管理员重置密码。管理员可以在"用户管理"中找到您的账号，点击"重置密码"。

### Q2: 为什么无法编辑某些工单？

**A**: 只有以下人员可以编辑工单：
- 工单创建者本人
- 当前负责人
- 系统管理员
- 相关部门成员（如研发部门成员可编辑研发相关工单）

### Q3: 工单状态有哪些？如何流转？

**A**: 工单状态包括：
- **创建/提交**：工单刚创建时的状态
- **处理中**：正在处理中的工单
- **已解决**：问题已解决
- **已关闭**：工单已关闭
- **暂时规避**：使用临时方案解决
- **未完成-客户原因**：因客户原因未完成
- **未完成-研发原因**：因研发原因未完成
- **未完成-生产原因**：因生产原因未完成
- **未完成-售后原因**：因售后原因未完成

状态流转：创建 → 处理中 → 已解决/已关闭/暂时规避

### Q4: 如何查看工单处理历史？

**A**: 
1. 打开工单详情页面
2. 查看"工单进度"区域
3. 时间线显示了所有状态变更记录
4. 包括：操作人、操作时间、状态变化、备注信息

### Q5: 如何导出特定条件的工单？

**A**:
1. 先使用搜索功能筛选出需要的工单
2. 点击"导出所有"按钮
3. 下载的 Excel 文件只包含筛选后的数据

### Q6: 系统支持哪些文件上传？

**A**: 系统支持上传图片和文档：
- 图片：jpg, jpeg, png, gif
- 文档：pdf, doc, docx, xls, xlsx
- 单个文件大小限制：10MB

### Q7: 如何更换头像？

**A**:
1. 点击右上角用户名
2. 选择"个人中心"
3. 点击头像区域
4. 选择新头像图片
5. 点击"保存"

### Q8: 工单优先级如何划分？

**A**:
- **P1（重大）**：系统瘫痪、核心业务中断，需要立即处理
- **P2（主要）**：重要功能异常，影响业务运行，需要尽快处理
- **P3（次要）**：一般功能异常，有替代方案，正常排队处理
- **P4（咨询）**：产品咨询、使用问题、优化建议等

### Q9: 为什么收不到系统通知？

**A**: 请检查以下几点：
1. 检查个人中心的邮箱是否正确
2. 检查邮箱是否将系统邮件标记为垃圾邮件
3. 联系管理员检查邮件服务配置

### Q10: 如何查看自己的工单统计？

**A**:
1. 点击左侧"工作看板"
2. 查看个人工单统计：
   - 待处理工单数
   - 本周新增工单数
   - 本月已解决工单数
   - 平均处理时长
