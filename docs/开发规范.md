### Python 代码规范

#### 1. 命名规范

| 类型 | 规范 | 示例 |

| 模块名 | 小写，下划线分隔 | `ticket_service.py` |

## 数据库规范

### 命名规范

| 对象 | 规范 | 示例 |
|------|------|------|
| 表名 | 小写，下划线分隔，复数 | `tickets`, `users` |
| 字段名 | 小写，下划线分隔 | `create_time`, `user_id` |
| 索引名 | `idx_表名_字段名` | `idx_tickets_status` |
| 外键名 | `fk_表名_字段名` | `fk_tickets_user_id` |

### 字段设计规范

```python
# 必须字段
id = db.Column(db.Integer, primary_key=True, autoincrement=True)
create_time = db.Column(db.DateTime, default=datetime.now)
update_time = db.Column(db.DateTime, default=datetime.now, onupdate=datetime.now)

# 软删除（推荐）
is_deleted = db.Column(db.Boolean, default=False)
delete_time = db.Column(db.DateTime, nullable=True)

# 状态字段使用枚举
status = db.Column(db.String(20), default='pending')  # pending, processing, completed

# 外键字段
user_id = db.Column(db.Integer, db.ForeignKey('users.id'))
```

### 索引设计原则

1. **主键自动创建索引**
2. **外键必须创建索引**
3. **频繁查询的字段创建索引**
4. **避免过多索引（一般不超过5个）**

```python
# 索引示例
class Ticket(db.Model):
    __tablename__ = 'tickets'
    
    id = db.Column(db.Integer, primary_key=True)
    status = db.Column(db.String(20), index=True)  # 频繁筛选
    priority = db.Column(db.String(10), index=True)  # 频繁筛选
    create_time = db.Column(db.DateTime, index=True)  # 频繁排序
    user_id = db.Column(db.Integer, db.ForeignKey('users.id'), index=True)  # 外键
    
    # 联合索引
    __table_args__ = (
        db.Index('idx_status_priority', 'status', 'priority'),
    )
```

## 安全规范

### 1. 输入验证

```python
from applications.common.utils.validate import str_escape

# 必须验证所有用户输入
def create_ticket():
    title = str_escape(request.json.get('title', ''))
    if not title or len(title) > 100:
        return fail_api(msg='标题不能为空且不超过100字符')
```

### 2. SQL注入防护

```python
# 使用 ORM，禁止字符串拼接 SQL
# 错误示例
query = f"SELECT * FROM tickets WHERE title = '{title}'"  # 危险！

# 正确示例
Ticket.query.filter(Ticket.title == title).all()
```

### 3. XSS防护

```python
# 输出时转义
from markupsafe import escape

# 模板中自动转义
# {{ user_input }}  # Jinja2 默认转义

# 手动转义
safe_output = escape(user_input)
```

### 4. 敏感信息保护

```python
# 使用环境变量
import os

SECRET_KEY = os.environ.get('SECRET_KEY')
DATABASE_PASSWORD = os.environ.get('DATABASE_PASSWORD')

# 禁止硬编码密码
# 错误示例
PASSWORD = '123456'  # 禁止！
```

## 性能规范

### 1. 数据库查询优化

```python
# 使用 select 减少字段
Ticket.query.with_entities(Ticket.id, Ticket.title).all()

# 使用 join 避免 N+1 查询
Ticket.query.options(joinedload(Ticket.user)).all()

# 使用分页
Ticket.query.paginate(page=1, per_page=10)
```

### 2. 缓存使用

```python
from applications.common.utils.cache import CacheManager

# 读取缓存
cache_key = f'ticket_{ticket_id}'
ticket = CacheManager.get(cache_key)

# 缓存不存在则查询并缓存
if not ticket:
    ticket = Ticket.query.get(ticket_id)
    CacheManager.set(cache_key, ticket, expired=300)
```
