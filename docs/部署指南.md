## 开发环境部署

### 1. 获取代码

```bash
# 克隆仓库
git clone https://gitee.com/pear-admin/pear-admin-flask.git

# 进入项目目录
cd pear-admin-flask
```

### 2. 创建虚拟环境

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. 安装依赖

```bash
# 安装项目依赖
pip install -r requirements.txt
```

### 4. 配置环境变量

创建 `.env` 文件：

```env
# 数据库配置
DATABASE_URI=mysql+pymysql://root:password@localhost:3306/pear_admin

# Redis配置
REDIS_URL=redis://localhost:6379/0

# 邮件配置
MAIL_SERVER=smtp.qq.com
MAIL_PORT=465
MAIL_USERNAME=your_email@qq.com
MAIL_PASSWORD=your_auth_code

# 密钥配置
SECRET_KEY=your-secret-key-here

# 调试模式
DEBUG=True
```

### 5. 初始化数据库

```bash
# 创建数据库表
flask db init
flask db migrate
flask db upgrade

# 初始化管理员账号
flask admin init
```

### 6. 运行项目

```bash
# 方式1：使用 Flask 命令
flask --app app.py run -h 0.0.0.0 -p 5000 --debug

# 方式2：直接运行
python app.py
```

访问地址：`http://localhost:5000`

默认账号：`admin` / `123456`

## 生产环境部署

### 1. 服务器准备

```bash
# 更新系统（CentOS）
sudo yum update -y

# 安装必要工具
sudo yum install -y git nginx supervisor

# 安装 Python 3.11
sudo yum install -y python3.11 python3.11-pip
```

### 2. 创建项目目录

```bash
# 创建项目目录
sudo mkdir -p /var/www/pear-admin
sudo chown -R $USER:$USER /var/www/pear-admin

# 克隆代码
cd /var/www/pear-admin
git clone https://gitee.com/pear-admin/pear-admin-flask.git .
```

### 3. 创建生产环境配置

创建 `.env.production` 文件：

```env
# 数据库配置（使用强密码）
DATABASE_URI=mysql+pymysql://pear_admin:StrongPassword123!@localhost:3306/pear_admin_prod

# Redis配置
REDIS_URL=redis://localhost:6379/0

# 邮件配置
MAIL_SERVER=smtp.qq.com
MAIL_PORT=465
MAIL_USERNAME=your_email@qq.com
MAIL_PASSWORD=your_auth_code

# 密钥配置（使用随机生成的强密钥）
SECRET_KEY=your-256-bit-secret-key-here

# 关闭调试模式
DEBUG=False

# 日志级别
LOG_LEVEL=INFO
```

### 4. 使用 Gunicorn 部署

安装 Gunicorn：

```bash
pip install gunicorn
```

创建启动脚本 `start.sh`：

```bash
#!/bin/bash

# 激活虚拟环境
source venv/bin/activate

# 启动 Gunicorn
gunicorn -w 4 -b 127.0.0.1:8000 app:create_app() \
    --access-logfile logs/access.log \
    --error-logfile logs/error.log \
    --log-level info \
    --daemon
```

赋予执行权限：

```bash
chmod +x start.sh
```

### 5. 配置 Nginx

创建 Nginx 配置文件 `/etc/nginx/conf.d/pear-admin.conf`：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 静态文件
    location /static {
        alias /var/www/pear-admin/static;
        expires 30d;
    }

    # 上传文件
    location /upload {
        alias /var/www/pear-admin/static/upload;
        expires 30d;
    }

    # 反向代理到 Gunicorn
    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # 错误页面
    error_page 500 502 503 504 /50x.html;
    location = /50x.html {
        root /usr/share/nginx/html;
    }
}
```

测试并重载 Nginx：

```bash
sudo nginx -t
sudo systemctl reload nginx
```

### 6. 配置 Supervisor

创建 Supervisor 配置文件 `/etc/supervisor/conf.d/pear-admin.conf`：

```ini
[program:pear-admin]
directory=/var/www/pear-admin
command=/var/www/pear-admin/venv/bin/gunicorn -w 4 -b 127.0.0.1:8000 app:create_app()
autostart=true
autorestart=true
user=www-data
stderr_logfile=/var/log/pear-admin/err.log
stdout_logfile=/var/log/pear-admin/out.log
```

启动 Supervisor：

```bash
sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start pear-admin
```

## 数据库配置

### MySQL 配置

```sql
-- 创建数据库
CREATE DATABASE pear_admin CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户
CREATE USER 'pear_admin'@'localhost' IDENTIFIED BY 'StrongPassword123!';

-- 授权
GRANT ALL PRIVILEGES ON pear_admin.* TO 'pear_admin'@'localhost';
FLUSH PRIVILEGES;
```

### 数据库迁移

```bash
# 生成迁移脚本
flask db migrate -m "Initial migration"

# 执行迁移
flask db upgrade

# 回滚迁移
flask db downgrade
```

## 常见问题

### 1. 安装依赖失败

**问题**：`pip install` 报错

**解决**：
```bash
# 更新 pip
pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 数据库连接失败

**问题**：`Can't connect to MySQL server`

**解决**：
```bash
# 检查 MySQL 服务
sudo systemctl status mysql

# 检查防火墙
sudo firewall-cmd --list-ports

# 检查用户权限
mysql -u root -p
SHOW GRANTS FOR 'pear_admin'@'localhost';
```

### 3. 静态文件无法访问

**问题**：404 错误

**解决**：
```bash
# 检查静态文件目录
ls -la /var/www/pear-admin/static

# 检查 Nginx 配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

### 4. 权限问题

**问题**：`Permission denied`

**解决**：
```bash
# 修改目录权限
sudo chown -R www-data:www-data /var/www/pear-admin

# 设置文件权限
sudo chmod -R 755 /var/www/pear-admin
```

### 5. 日志查看

```bash
# 查看应用日志
tail -f /var/log/pear-admin/out.log

# 查看错误日志
tail -f /var/log/pear-admin/err.log

# 查看 Nginx 日志
tail -f /var/log/nginx/error.log
```

## 性能优化

### 1. 数据库优化

```sql
-- 添加索引
CREATE INDEX idx_ticket_status ON ticket(status);
CREATE INDEX idx_ticket_priority ON ticket(priority);
CREATE INDEX idx_ticket_create_time ON ticket(create_time);
```

### 2. Nginx 优化

```nginx
# 启用 gzip
gzip on;
gzip_types text/plain text/css application/json application/javascript;

# 启用缓存
location /static {
    expires 30d;
    add_header Cache-Control "public, immutable";
}
```

### 3. Gunicorn 优化

```bash
# 根据 CPU 核心数调整 worker 数量
gunicorn -w $(($(nproc) * 2 + 1)) -b 127.0.0.1:8000 app:create_app()
```
