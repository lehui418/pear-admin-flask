# 网安工单系统页面样式优化说明

## 优化内容

### 1. 样式统一化
- 提取了 `edit.html` 和 `view.html` 的共同样式到 `common-styles.css`
- 统一了两个页面的布局风格和视觉效果
- 保持了各页面特有功能的样式独立性

### 2. 布局优化
- **两列布局**: 采用 49% 宽度的响应式两列布局
- **分组样式**: 统一的表单分组样式，包含背景色、边框和阴影
- **响应式设计**: 在小屏幕设备上自动切换为单列布局

### 3. 样式特性

#### 通用样式
- 统一的表单元素样式（输入框、标签、按钮等）
- 一致的颜色标签系统（红、橙、绿、蓝、灰）
- 统一的威胁等级指示器
- 网安特色的安全等级样式

#### 编辑页面特有
- 固定底部操作按钮（保存、取消）
- 下拉菜单优化样式
- 特殊字段的样式增强（CVSS评分、IOC指标等）
- 工单超时提醒样式

#### 查看页面特有
- 详情显示样式
- 描述内容的特殊格式化
- 居中的返回按钮

### 4. 文件结构

```
templates/system/ticket/
├── edit.html          # 编辑页面（已优化）
├── view.html          # 查看页面（已优化）
└── README.md          # 本说明文件

static/system/ticket/
└── common-styles.css  # 通用样式文件
```

### 5. 响应式特性

- **桌面端**: 两列布局，固定底部按钮
- **移动端**: 单列布局，相对定位按钮
- **自适应宽度**: 表单元素在小屏幕上占满宽度

### 6. 使用方法

两个页面都会自动引入通用样式文件：
```html
<link rel="stylesheet" href="{{ url_for('static', filename='system/ticket/common-styles.css') }}">
```

### 7. 维护建议

1. **新增通用样式**: 添加到 `common-styles.css`
2. **页面特有样式**: 保留在各自的 `<style>` 标签中
3. **响应式调整**: 在 `@media` 查询中进行
4. **颜色主题**: 统一使用定义好的颜色变量

### 8. 兼容性

- 支持现代浏览器的 CSS3 特性
- 兼容 Layui 框架的样式系统
- 保持与原有功能的完全兼容
